# BBEDUAlertForAndroid

### 앱 기능 설명 - 로딩 화면
'''
이 화면에서는 Google Firebase 에 접속하여 프로그램 이용 가능 여부 또는 버전 정보와 사용자 로그인 확인 및 사용자 데이터를 불러오게 됩니다.
로그인은 Google OAuth를 사용합니다.
